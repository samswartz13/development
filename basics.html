HTML Display Attributes

Block Elements

These elements stretch from one edge of the page to the other. They respect margin, height & padding

.div {
  background-color: aquamarine;
  height: 100px;
  /* display is automatically set to block */
  /* Default width is the full width of the screen */
}

Real world use: most layouts are built with divs - these are the blocks that usually hold all your content

Setting width on block level elements will NOT make them flow next to one another. See inline-block, inline, or float for that behavior

.divShort {
  background-color: aquamarine;
  height: 100px;
  width: 50px;

}
.divShort {
  background-color: aquamarine;
  height: 100px;
  width: 50px;

}

Inline Elements

Inline elements only fit to the content that's inside of them. Unlike divs, inline elements do not extend to the full width of the screen and can easily go next to one another. Inline elements also don't respect margin & height.

My awesome span  My second awesome span 
.basicSpan {
  background-color: darkturquise;
  height: 100px;	/*This does nothing */
  /* display is automatically set to inline */
}

Real world use: these can be really good for adding in content inline with text. A link for example, is an inline element. The text for "Real world use" in this section is also made with a span ;)

Inline Block Elements

Inline block elements, like spans, fit their width to the content that's inside of them. Inline Block elements also DO respect margin and height.

Inline Block
Inline Block
.inlineBlock {
  background-color: cornflowerblue;
  height: 50px;
  display: inline-block;
}

Real world use: This is a pretty powerful layout tool - anytime you want content side by side, inside block is an option

Show/Hide with Display Properties

Display None Property

Display None is a CSS Property that can be set on any HTML element. When display none is applied, it's as if the content was never there. Any parent or sibling elements end up positioned as if the there was nothing else on the page

Inline Block
Inline Block
.inlineBlock {
  background-color: cornflowerblue;
  height: 50px;
  display: inline-block;
}
.displayNone {
  display: none;
  /* Open up dev tools to find the missing div */
}

Visibility Hidden Property

Visibility Hidden is a css property that also hides whatever element it's applied to. What makes it different from Display None is that the parent and sibling elements still act as if the object was there.

Inline Block
Inline Block
 
.inlineBlock {
  background-color: cornflowerblue;
  height: 50px;
  display: inline-block;
}
.visibilityHidden {
  visibility: hidden;
  /* Open up dev tools to find the hidden div */
}

Clearing Floats

Let's say we wanted to make a floated nav bar. We use floats because we don't want any space between our divs, but we also don't want our content to wrap around our floats anymore. This is when you would use the css clear property.

CSS Position Properties

Position Static

Static is the default position of an HTML element. Top, right, left, and bottom properties will have no effect on a static element.

.div {
  top: 0;
  right: 0;
  left: 0;
  bottom: 0;
}
  /* no effect since div is position static */
}

Position Relative

Setting top, bottom, left or right on an element along with position relative will allow you to move the element relative to its original position. Sibling and parent elements will not be effected by any change in position.

Inline Block
Inline Block
.inlineBlock {
  background-color: cornflowerblue;
  height: 50px;
  display: inline-block;
}

.inlineBlock-Relative {
  background-color: aqua;
  height: 50px;
  display: inline-block;
  position: relative;
  top: 30px;
  left: -15px;
}

Position Absolute

Setting top, bottom, left or right on an element along with position absolute will allow you to move the element relative to its parent. Sibling and parent elements will flow as if the position absolute element were not there.

Inline Block
Absolute
.parent {
  background-color: alice-blue; /* lt blue */
  position: relative;
  /* position relative contains 
  position absolute child */
}


.inlineBlock {
  background-color: cornflowerblue;
  height: 50px;
  display: inline-block;
}

.absolute {
  background-color: aqua;
  height: 50px;
  position: relative;
  bottom: 0px;
  left: 15px;
}

Position Fixed

Setting top, bottom, left or right on an element along with position fixed will allow you to move the element relative to the BROWSER and keeps it pinned to it's given position. This means that scrolling will not move the element offscreen. Sibling and parent elements will flow as if the position fixed element were not there.

The navigation bar on this site is an example of position fixed ;) 
.fixed {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  height: 50px;
}

Z Index

Z Index makes something appear closer to the screen viewer. It allows you to layer HTML elements. You can set a Z Index on any element with a position attribute. An element with it's origninal position static property will not respect z-index.

Inline Block
Inline Block
.inlineBlock-zIndex {
  background-color: coral;
  height: 50px;
  display: inline-block;
  position: relative; /*Position relative is needed */
  z-index: 200;
}
